3. POST /books

Description: Create a new book

URL: /books

Body Parameters:

  • title: (string) The title of the book
  • description: (string) A short description of the book
  • rating: (integer) A rating for the book (e.g., 1-5)
  • image: (string) URL or path to the book's image
  • user_id: (integer) The ID of the user who added the book

Example Request: POST /books

        {
            "title": "New Book Title",
            "description": "A detailed description of the new book.",
            "rating": 5,
            "image": "image.png",
            "user_id": 1
        }

Example Response:

        {
            "message": "Book created",
            "bookId": 10
        }

4. PUT /books/:id

Description: Update an existing book

URL: /books/:id

Body Parameters:

  • title: (string) The title of the book
  • description: (string) A short description of the book
  • rating: (integer) A rating for the book (e.g., 1-5)
  • image: (string) URL or path to the book's image
  • user_id: (integer) The ID of the user who added the book

Example Request: PUT /books/10

        {
            "title": "Updated Book Title",
            "description": "Updated description of the book.",
            "rating": 4,
            "image": "updated.png",
            "user_id": 1
        }

Example Response:

        {
            "message": "Book updated",
            "result": 1
        }

5. DELETE /books/:id

Description: Delete a book by ID

URL: /books/:id

Query Parameters: None

Example Request: DELETE /books/9

Example Response:

        {
            "message": "Book with ID 9 has been deleted successfully."
        }
